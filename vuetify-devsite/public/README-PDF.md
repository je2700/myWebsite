# PDF Lebenslauf hinzufügen

## So fügst du deine Lebenslauf-PDF hinzu:

1. **Benenne deine PDF-Datei um zu**: `lebenslauf.pdf`

2. **Kopiere die Datei in diesen Ordner**: 
   ```
   /workspaces/myWebsite/vuetify-devsite/public/
   ```

3. **Oder per Terminal**:
   ```bash
   cp /pfad/zu/deiner/datei.pdf /workspaces/myWebsite/vuetify-devsite/public/lebenslauf.pdf
   ```

4. **Die Datei wird automatisch verfügbar** unter:
   - Lokal: http://localhost:5173/lebenslauf.pdf
   - Produktiv: https://je2700.github.io/lebenslauf.pdf
